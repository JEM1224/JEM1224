1. java에서 int와 Integer의 차이점  
 - Number 클래스는 `java.lang`에 있는 추상 클래스이고 Integer는 Number를 상속 받아 구현한 클래스
 - Integer,Byte,Float 등 클래스들은 원시타입을 wrap 해주는 wrap 클래스이다.
 2. box와 unbox
 - box :  원시 타입을 Object로 포장
 - unbox : Object을 원시 타입으로 벗겨냄
